Easy Instagram Feed plugin allows to add the instagram feeds on your wordpress site. 

== Description ==

Easy Instagram Feed plugin allows to fetch the instagram feeds in your wordpress site. Just add the shortcode [easyinstagramfeed] in the normal wordpress page/post inorder to get the feeds.


= Features =
* Easily add Instagram Feed using the shortcode [easyinstagramfeed]
* Add multiple users feeds.
* Option for configuring instagram feed section.
* Lots More to Come.

Here is the link to [Documentation](http://webriti.com/documentation-for-easy-instagram-feed-plugin-lite-version/)

In case you face any problem, contact us via the [Forums](http://wordpress.org/support/plugin/easy-instagram-feed). 

= RoadMap =
This is a early version and we have lots of features under development. Here is a tentative RoadMap

* Sorting of Photos.
* Setting for selecting number of photos to display.
* Pagination, feed column layout, setting resolution of images.
* Custom css box to use the feeds you own etc etc.


== Installation ==

1. Download Easy Instagram Feed plugin.
2. Upload the easy-instagram-feed folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the 'Plugins' menu in WordPress and Enjoy.
